# Documentazione progetto

Questa cartella contiene la documentazione operativa del progetto.

## Contenuto

- `overview.md`: panoramica generale del progetto.
- `user-guide.md`: guida per utenti finali.
- `admin-guide.md`: guida per amministrazione e configurazione.
- `faq.md`: domande frequenti.
- `changelog.md`: modifiche rilevanti nel tempo.

## Regola

Ogni modifica che impatta utenti, cliente o amministrazione deve aggiornare almeno uno dei file in `docs/`.
