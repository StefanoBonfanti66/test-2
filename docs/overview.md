# Panoramica

## Obiettivo

**Race Planner** è un'applicazione web per la pianificazione e gestione di gare sportive (triathlon, corsa, nuoto, ciclismo).

## Utenti coinvolti

- **Atleta**: utente finale che visualizza e pianifica le proprie gare
- **Allenatore**: gestisce atleti e piani di allenamento
- **Amministratore**: gestisce la piattaforma, utenti e configurazioni globali

## Stack tecnico

| Componente | Tecnologia |
|------------|------------|
| Frontend | React 19 + TypeScript + Vite |
| Stili | TailwindCSS 4 |
| Routing | React Router 7 |
| Backend/Auth | Supabase |
| Database | PostgreSQL con RLS |
| Deploy | Vercel |
| CI/CD | GitHub Actions |

## Funzionalità pianificate

- Visualizzazione calendario gare
- Profilo atleta con statistiche
- Gestione piani di allenamento
- Integrazioni con servizi di tracking

## Integrazioni

- **Supabase**: Authentication, Database Row Level Security, Storage
